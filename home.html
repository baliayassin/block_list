Get Started | API Documentation | block list

The amazon books API allows apps to interact directly with the block list of books .
Some typical use cases include: Get all books, Get user profile data by id, Get user books by date.

Introduction

The Amazon Books API created to give details about block list of books.
All the requests should be in GET / POST / PUT Method .
The result will return in JSON File.
If your URL request is wrong you will forward to 200 Error page.

Get All Books

You can find all the books in amazon, by entering the following URL:

https://block-lists.herokuapp.com/getAll Method: GET
